# Changelog

All notable changes to PlantWise will be documented in this file.

## [1.0.0] - 2024-12-XX

### Added
- Initial release of PlantWise
- Real-time air quality detection
- AI-powered plant recommendations
- User authentication system
- Interactive dashboard
- Plant saving functionality
- Community features
- Image upload for space analysis

### Features
- Pollution-based plant filtering
- Location-based AQI monitoring
- Personalized plant suggestions
- Environmental impact tracking
- Social community integration

### Technical
- React 18 with TypeScript
- TailwindCSS design system
- Supabase backend integration
- Framer Motion animations
- React Query state management
