Pollution Aware Smart Plant Recommendation System

Project Description
This AI-based system recommends pollution controlling plants based on air quality levels.

Features
- Detect pollution level
- Recommend suitable plants
- Easy interface

Technologies Used
-HTML, CSS, JavaScript
-React (Frontend)
-Flask (Backend)
-Machine Learning (Image & Data Analysis)
-AQI / Pollution Dataset
-Figma (UI/UX Design)

How it Works
1. User enters pollution level
2. System analyzes the data
3. Recommended plants are displayed
